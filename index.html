How to Set Up Google Sheets Integration

  1. Create a Google Sheet with columns: Timestamp, Name, DS Name, Mobile, Role
  2. Go to Tools > Script editor and create a new Apps Script
  3. Copy the provided script code and save it
  4. Deploy as a web app with "Anyone" access
  5. Copy the web app URL and paste it in the field below
